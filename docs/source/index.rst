Introduction
------------

Docify helps you write documents in different formats from a **Python object**

..

   **Note:** It's not a document *converter*\ , neither a *documentation* generator, nor a document *parser*


Installation
------------

.. code-block:: bash

   pip install docify


Usage & Examples
----------------

** `Find quickstart and more examples here <https://github.com/rapidstack/docify/tree/master/examples>`_ **
